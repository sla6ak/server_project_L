Lost Space API Server

This is the API server for Lost Space application. The server provides authentication and real-time communication services.

Available Endpoints:

POST /auth/signup

Register a new user account

POST /auth/login

Authenticate user and get access token

GET /auth/current

Get current user information

POST /auth/logout

Logout current user

WebSocket Support

This server also provides WebSocket endpoints for real-time communication.

WebSocket URL: ws://[server-address]/
